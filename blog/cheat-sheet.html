For loop in Ruby

Used for iterating through a known number of steps.

for i in 0..10
    puts "the current loop number is #{i}"
end

# or,

for element in array
    puts "The current element is: #{element}"
end

For loop in Javascript

Used for iterating through a known number of steps.

for (var i = 0; i < 10; i++){
    console.log( "the current loop number is: " + i);
}

// or,

for each (var element in array) {
    console.log( "The current element is: " + element);
}

While loop in Ruby

Used for conditional looping. The while keyword allows you to run a loop while a given condition is true. The code block must be followed by an end keyword. 

n = 1
while n < 10
    puts n
    n = n + 1
end

While loop in JavaScript

Used for conditional looping. The keyword while is followed by an expression in parentheses, which determins when the loop will finish. 

var n = 1
while (n < 10) {
    console.log(n);
    n = n + 1;
}

Operating on each member of an Array in Ruby

Used for performing some function on each element in an array.

names = ["peter", "paul", "mary"]

names.each {| name | puts "#{name} is in the band!"}

Operating on each member of an Array in Javascript

Used for performing some function on each element in an array.

var names = ["peter", "paul", "mary"];

function printNames (element, index, array) {
    console.log(element + " is in the band!");
}

names.forEach(printNames);
