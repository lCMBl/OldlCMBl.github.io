CSS positioning

or, "I'm still having problems with my nav bar"

Feb 15th, 2015

So, If you're like me, (unlikely, but possible) then CSS positioning has been a huge headache for you. Specifically, getting elements to stay at their respectve spots on the page while the page is resized or zoomed has proven difficult for me. Fear not though! Despite my work in progress implementation of css positioning, I have picked up enough to be useful. Specifically, I am now capable of explaining the differences between absolute, relative, static, and fixed positions to all of you lucky people.

First things first, Static positioning is about as basic as you can get. Pretty much every element defaults to static positioning, and all it means is that elements will be rendered in their normal spot. Easy, right?

The next type of positioning is much more powerful, but can also be tricky to use. This is of course, the relative position scheme. Basically, setting an element to relative positioning means that an element will be positioned relative to where that element would have been if it was positioned statically. So, an element with a relative position and a left value of 10 px will be offset 10 px to the right of where it would normally be.

One of the more powerful types of positioning is the absolute position, which positions an element relative to the next parent element that is either relatively or absolutely positioned. If no such element exists, the element is positioned relative to the html document itself. Basically, the absolute positioning lets you place the element precisely where you want it. (elements positioned with absolute positioning)

Fixed position is useful for things like header or nav bars that need to be accessible at all times. Rather than positioning relative to anything in the html document itself, fixed position elements position themselves relative to the view port. This means that they will always occupy the same space on the screen, regardless of the elements on the page or scrolling.

That's all for now! - Christian
